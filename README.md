# Elisé (LINE Chat Bot) #

[![Build Status](https://api.travis-ci.org/Diptags/Elise-LineChatBot.svg?branch=master)](https://travis-ci.org/Diptags/Elise-LineChatBot/)

Elisé saat ini masih dalam pengembangan lebih lanjut
